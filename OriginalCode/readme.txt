README

Source code for manuscript "Variable selection – a review and recommendations for the practicing statistician"
by Georg Heinze, Christine Wallisch, and Daniela Dunkler.

For questions, comments or remarks about the code please contact Georg Heinze (georg.heinze@meduniwien.ac.at).

The code has been written using R version 3.3.2 (platform: x86_64-w64-mingw32/x64, 64-bit) with package versions doParallel_1.0.10, foreach_1.4.3, mvtnorm_1.0-5, shrink_1.2.1, and survival_2.40-1.

To reproduce the results presented in the manuscript, run the file Code.R in each folder. 

Description of the data set in case study 1 can be found at:
https://ww2.amstat.org/publications/jse/datasets/fat.txt
We excluded one patient because of his low height, and we converted the units of some variables.

The data set in case study 2 is described in the package shrink.
